## 异常处理


### 下载安装

~~~go
go get -u github.com/deatil/go-exception
~~~


### 使用

~~~go
package main

import "github.com/deatil/go-exception/exception"

exception.
    Try(func() {
        panic("exception error")
    }).
    Catch(func(e exception.Exception) {
        fmt.Println(e.GetMessage())
    })

~~~
