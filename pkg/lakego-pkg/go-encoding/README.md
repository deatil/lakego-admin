## 编码算法


### 项目介绍

*  常用的编码算法合集


### 使用方法

~~~go
package main

import (
    "fmt"
    "github.com/deatil/go-encoding/encoding"
)

func main() {
    // Base64 编码后结果
    base64Data := encoding.Base64Encode("useData").

    fmt.Println("Base64 编码后结果：", base64Data)
}

~~~
