## 集合处理


### 项目介绍

*  常用的集合处理


### 下载安装

~~~go
go get -u github.com/deatil/go-collection
~~~


### 使用

~~~go
package main

import (
    "fmt"
    "github.com/deatil/go-collection/collection"
)

func main() {
    collection.Collect([]string)
}

~~~
